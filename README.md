
# Spring Boot REST API with MySQL and JWT Authentication

Proyek ini adalah contoh implementasi REST API menggunakan Spring Boot, MySQL sebagai database, dan JWT untuk autentikasi.

## 🚀 Fitur

- CRUD sederhana untuk entitas User
- Login dan Registrasi menggunakan JWT
- Integrasi dengan MySQL Database
- Spring Security Config

## ⚙️ Konfigurasi `application.properties`

```properties
spring.datasource.url=jdbc:mysql://localhost:3306/your_database_name
spring.datasource.username=your_username
spring.datasource.password=your_password
spring.jpa.hibernate.ddl-auto=update
spring.jpa.show-sql=true
spring.jpa.properties.hibernate.dialect=org.hibernate.dialect.MySQL5Dialect
```
